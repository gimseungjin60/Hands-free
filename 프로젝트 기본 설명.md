1. 저항(Resistor) 색 띠 읽기 — 학생이 가장 헷갈리는 부분

이 보드의 저항은 5색 띠 1% 정밀저항이다.


띠 의미: 1·2·3번 띠 = 숫자, 4번 띠 = 배수, 5번 띠 = 허용오차
색–숫자: 흑0 · 갈1 · 적2 · 등(주황)3 · 황4 · 녹5 · 청6 · 자7 · 회8 · 백9
배수(4번 띠): 금 ×0.1 · 흑 ×1 · 갈 ×10 · 적 ×100 · 등 ×1000
허용오차(5번 띠): 갈색 = ±1%


읽는 방향 (핵심 혼동 포인트)


허용오차 띠(여기선 갈색 = ±1%)가 끝(뒤)에 온다. 그 반대쪽 끝부터 읽는다.
양 끝이 모두 갈색이라(예: 10kΩ = 갈-흑-흑-적-갈) 헷갈릴 때는, 색을 해독하려 하지 말고 위치별 정답 값을 조회해 확인한다.
저항은 극성이 없어 전기적으로는 방향이 무관하다. 단, 값 식별과 보드 가독성을 위해 읽는 방향을 맞춘다.



2. 극성·방향이 중요한 부품 (다리 길이 / 띠 / 홈)


LED (D1, D4, D6~D13): 긴 다리 = +(애노드). 보드의 사각 패드 / + 표시 쪽.
전해 콘덴서 (C2, C4, C21~C24): 짧은 다리 = -(음극). 몸통 흰 띠가 있는 쪽이 -.
다이오드 (D2 1N4148, D3 1N5819, D5): 몸통 띠 쪽 = 캐소드(-). 보드 띠 표시와 일치.
IC / EEPROM (U4 AT24C02, U5 MAX232): 홈(notch)·점 방향을 보드 홈 표시와 일치.
어레이 저항 (AR1 1kΩ×4, AR2 330Ω×8): 점(•) 표시 = 1번 핀. 보드 표시와 맞춤.
트랜지스터 (TR5 DTC113ZSA): 평평한 머리 면을 보드 그림과 일치.
7-세그먼트(U6), 박스헤더(J12 2×5, J13 2×3), 커넥터(J2 등): 홈·삼각형 방향 일치.
극성 없음: 세라믹·탄탈 콘덴서, 일반 저항.



3. 납땜 기본


5단계: 가열(2~3초) → 솔더 공급 → 솔더 당김 → 인두 당김 → 식히기(입으로 불지 말 것).
좋은 납땜: 매끈한 원뿔 형태. 불량: 냉납(cold joint), 쇼트(short), 과열, 납량 부족.
노하우: 다리 많은 부품은 한 다리만 먼저 납땜 → 뒤집어 밀착·정위치 확인 → 나머지 납땜. 고정이 어려우면 대각선부터 하나씩.
안전: 인두기·거치대 위치 항상 의식(좌우 학생 것까지), 작업영역 정리.
장비: FX-600 인두기 약 420℃, 팁 자주 닦기, 거치대 몸체 잡지 않기.



4. 전원부 검증


LM7805(U2) 출력에서 +5V / GND를 측정해 전원이 정상인지 확인한다.

7. 조교 동작 원칙 (요약)


부품 값·극성·위치는 반드시 데이터로 조회해 답하고, 모르면 추측하지 말고 모른다고 한다.
학생이 자주 막히는 저항 색 띠 방향과 극성 부품 다리는 묻기 전에 먼저 짧게 경고한다.
답에 이유 한 문장을 덧붙인다. (예: "허용오차 띠가 끝이라 반대쪽부터 읽어.")
음성으로 듣기 좋게 짧고 명확하게, 한국어로 답한다.
